This is the documentation for the {\ttfamily sprint\+\_\+3} package, which includes a ROS 2 node designed for detecting cylindrical objects using Laser\+Scan data.\hypertarget{index_author_sec}{}\doxysection{Author}\label{index_author_sec}
This package was developed by Jack Gruber.\hypertarget{index_usage_sec}{}\doxysection{Usage}\label{index_usage_sec}
The main class is {\ttfamily \mbox{\hyperlink{classCylinderDetectionNode}{Cylinder\+Detection\+Node}}}, which subscribes to Laser\+Scan messages and processes the data to detect cylindrical objects.\hypertarget{index_features_sec}{}\doxysection{Features}\label{index_features_sec}

\begin{DoxyItemize}
\item Subscribes to Laser\+Scan data.
\item Converts polar coordinates to Cartesian coordinates.
\item Clusters points to identify potential cylindrical objects.
\item Checks for cylindrical shapes based on clustered points.
\item Logs detected cylinders with their midpoints.
